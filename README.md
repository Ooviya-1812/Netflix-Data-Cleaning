# Data Cleaning Project using Microsoft Excel

## Project Overview
This project focuses on cleaning and preparing a raw dataset using Microsoft Excel. The objective was to improve data quality by handling missing values, removing duplicate records, and standardizing the dataset for accurate analysis and reporting.

## Objectives
- Improve the quality and consistency of the dataset.
- Prepare clean and structured data for analysis.
- Apply industry-standard data cleaning techniques using Microsoft Excel.

## Tools Used
- Microsoft Excel

## Data Cleaning Tasks Performed
- Removed duplicate records.
- Identified and handled missing (blank) values.
- Standardized date formats (DD-MM-YYYY).
- Standardized column formats and data types.
- Verified data accuracy and consistency.
- Prepared the dataset for further analysis.
